Key design principles
No magic numbers · Every constant in Config struct
Clean separation · Simulation knows nothing about rendering
Zero GC pressure · Rust ownership model, shared buffers
Deterministic · Same seed always produces the same world
Evolutionary costs · Every advantage has a tradeoff
Arc field of view · No animal has 360° vision
Performance targets
60 fps · At 2000 agents on 4000×2000 world
Zero GC spikes · Rust simulation core, shared buffers
O(n²) → O(n log n) · Spatial grid index when needed
Arc scan · ~50% reduction in effective scan area vs circle
Stress tested · Rust/WASM vs TypeScript comparison complete
